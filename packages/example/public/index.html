About This Demo

This runs the Glulxe interpreter via @bodar/wasiglk, using a Web Worker and JSPI for async I/O.

Try commands like: look, go north, inventory, help

Requirements

Chrome: JSPI enabled by default in Chrome 131+, or enable via chrome://flags/#enable-experimental-webassembly-jspi

Firefox: Enable javascript.options.wasm_js_promise_integration in about:config
